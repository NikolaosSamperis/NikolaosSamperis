## 🚀 Selected Projects

### 🔷 [PlaqueMS Project](https://github.com/NikolaosSamperis/PlaqueMS-Platform)
*A secure web platform for multi-cohort omics analysis with embedded predictive modelling (MSc Thesis Project)*  
- Developed using Django (Python) for the backend and JavaScript, HTML, and CSS for the frontend, with a hybrid database architecture combining MySQL and Neo4j.  
- Integrated multi-modal proteomics and clinical metadata across three independent cohorts.
- Supports interactive exploration, network visualisation (Cytoscape), and embedded ML models (SVM, Random Forest, Elastic Net Regression) optimised via a multi-objective evolutionary algorithm.

### 🤖 [GenAI Natural-Language Data Assistant](https://github.com/NikolaosSamperis/ads-programmer-coding-assessment/tree/main/question_6_genai)
*An LLM-powered assistant that lets users explore datasets through plain-English questions — no code required*
- Built with **Python, LangChain, and the Claude API**, translating natural-language questions into executable `pandas` queries.
- Implemented **prompt engineering** and **structured output (Pydantic)** to ensure reliable, validated, and reproducible responses.
- Removes the need for users to write code in order to explore and interrogate a dataset, lowering the barrier to data-driven decision-making.
<br>

### 🧠 [CNN Image Classifier](https://github.com/NikolaosSamperis/CNN-RetinalDisease-Pipeline)
*Deep learning-based image classification using transfer learning*  
- Transfer learning using **EfficientNetB0** and **DenseNet-121** with a custom classification head and selective fine-tuning.  
- Applied data augmentation, focal loss, and validation strategies to improve model sensitivity on an imbalanced dataset.
<br>

### 📊 [Statistical Modelling: Screening Uptake](https://github.com/NikolaosSamperis/breast-cancer-screening-statistical-analysis)
*Reproducible statistical analysis of factors associated with health-screening uptake*
- End-to-end statistical workflow in **R**: data cleaning, validation, exploratory analysis, and multivariable logistic regression modelling.
- Investigated demographic and behavioural factors across **12 teams (528 individuals)**.
- Identified significant predictors while assessing missing-data patterns and potential heterogeneity between groups.
<br>

### 🧬 [Genotype2Phenotype Dashboard](https://github.com/NikolaosSamperis/Genotype2Phenotype)
*Interactive RShiny dashboard for exploring data relationships at scale*  
- MySQL-backed dashboard with **UMAP-driven clustering** and interactive visualisation.  
- Robust preprocessing and dynamic filtering across large datasets (156,000+ files).
<br>

### 🧪 [RNA-seq Analysis Pipeline](https://github.com/NikolaosSamperis/Reconstruction-of-RNA-seq-analysis)
*Reconstructed and benchmarked an analysis pipeline to assess reproducibility and methodological transparency*
- Preprocessing including quality assessment with `FastQC`, alignment using `STAR`, and quantification with `featureCounts`.
- Differential expression and enrichment analysis in R using Bioconductor packages (e.g. `DESeq2`, `clusterProfiler`).  
- Generated PCA, heatmaps, and volcano plots for exploratory and statistical analysis.
